# Real Memory Browser Agent Privacy Policy

Effective date: June 29, 2026

Real Memory Browser Agent is published by Reallexi LLC. This policy explains what data the browser extension can access, how it is used, and how users can control it.

## Data The Extension Handles

The extension may handle the following data when a user enables features or runs a task:

- User prompts, chat messages, workflow instructions, and attached screenshots or files.
- Active tab context, including URL, title, visible page text, headings, links, metadata, form-field labels, and screenshot evidence when requested.
- Extension settings such as selected model flow, endpoint URL, model names, timeout settings, and provider credentials entered by the user.
- Runtime records such as sessions, activity logs, artifacts, screenshots, JSON evidence, connection-test results, queue state, and error messages.

## How Data Is Used

Data is used only to provide the extension features requested by the user:

- Chat with the selected model provider.
- Inspect the active browser tab and summarize visible page context.
- Capture screenshots and evidence for review.
- Run browser-safe actions such as opening tabs, clicking visible controls, filling simple fields, downloading files, or saving artifacts.
- Test provider connections and refresh available models.
- Keep local session history and activity records for the user to review.

## Local Storage

Real Memory Browser Agent stores extension state in extension-owned browser storage, including IndexedDB and `chrome.storage.local`. This can include settings, sessions, messages, activity records, artifacts, screenshots, and provider credentials entered by the user.

The extension does not use cookies for provider configuration, chat history, or activity state.

## Network Sharing

When the user sends a request, the extension may send the prompt, relevant page context, screenshots, attachments, selected settings, and credentials required for authentication to the model provider or endpoint selected by the user. Supported destinations may include local model servers, enterprise endpoints, cloud AI APIs, Real Memory services, OpenAI-compatible gateways, or exact custom endpoint URLs configured by the user.

Provider availability, logging, retention, and training practices are controlled by the selected provider or endpoint. Users should review the privacy terms of any provider they choose to connect.

## Data Sale And Advertising

Reallexi LLC does not sell, rent, or trade personal data collected through Real Memory Browser Agent. The extension does not use personal data for advertising profiles or ad targeting.

## Data Sharing

Data is shared only when needed to operate the extension features requested by the user, including:

- Sending requests to the user-selected provider or endpoint.
- Using browser APIs required for storage, active-tab inspection, screenshots, downloads, and side-panel operation.
- Complying with legal, security, abuse-prevention, or platform-review requirements when required.

## User Controls And Deletion

Users can clear extension sessions, artifacts, activity records, and settings inside the extension where controls are provided. Users can also remove all locally stored extension data by uninstalling the extension or clearing the extension's site and storage data from the browser.

Data sent to a user-selected provider is controlled by that provider's retention and deletion process.

## Permissions

The extension requests browser permissions needed for its visible features, including side-panel operation, storage, active-tab inspection, tab metadata, screenshots, downloads, scripting for visible page actions, and local-provider network handling. These permissions are used to complete user-requested browser-agent tasks and to keep evidence available for review.
