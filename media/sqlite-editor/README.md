# Reallexi SQLite Editor Public Media

This folder is generated by `npm run package` from the SQLite editor repository.

- Icons: `icons/`
- Screenshots and demo media: `screenshots/`
- Store policies and compliance files: `policies/`
- Public README, changelog, license, and package metadata: `public-files/`
- VSIX package artifact: `packages/sqlite-free-editor-0.1.21.vsix`

- Chrome packed ZIP: `browser-extensions/sqlite-free-editor-chrome-0.1.21.zip`
- Edge packed ZIP: `browser-extensions/sqlite-free-editor-edge-0.1.21.zip`
- Chrome unpacked extension: `browser-extensions/unpacked/chrome`
- Edge unpacked extension: `browser-extensions/unpacked/edge`
