# Store Image Assets

These images are generated by `npm run store:assets` and refreshed by `npm run package`.

Official size coverage:

- Chrome screenshots: 1280x800 and 640x400.
- Chrome small promotional image: 440x280.
- Chrome marquee promotional image: 1400x560.
- Microsoft Edge logo: 300x300.
- Microsoft Edge screenshots: 1280x800.
- Microsoft Edge alternate screenshots: 640x480.
- Microsoft Edge small promotional tile: 440x280.
- Microsoft Edge large promotional tile: 1400x560.
