# Reallexi SQLite Editor Privacy Policy

Reallexi SQLite Editor is owned by Reallexi LLC.

The extension does not collect, sell, share, or broker personal data. It runs inside Visual Studio Code and works with SQLite database files the user opens or creates in their workspace.

Database contents, SQL queries, imports, exports, backups, save operations, and file paths remain in the user's VS Code environment and selected filesystem locations. The extension does not transmit database contents or usage activity to Reallexi LLC.

The extension uses packaged code and packaged dependencies. It does not load or execute remote JavaScript or WebAssembly.

Support: https://github.com/ReallexiLLC/sqlite-editor/issues
