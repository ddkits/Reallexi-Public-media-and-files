# Source And Attribution Policy

Mawajez uses public RSS, Atom, Google News RSS, and YouTube feed endpoints from publishers and links readers back to the original source for the full story.

The extension displays headline metadata, source name, category, timestamp, and original or Mawajez URL. Mawajez does not claim ownership of publisher articles or videos.

If a publisher feed becomes unavailable, the extension falls back to the most recent cached Mawajez API snapshot stored in the browser.
