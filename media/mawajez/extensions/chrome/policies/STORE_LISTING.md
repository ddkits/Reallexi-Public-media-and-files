# Store Listing Notes

Name: Mawajez News Hub

Short description: Multilingual global news search, source filters, favorites, and session restore from Mawajez public feeds.

Single purpose: Provide quick access to Mawajez public news feeds with local favorites and browser session continuity.

Data handling: Favorites and settings remain in browser storage. The extension fetches public JSON from `https://mawajez.com/api/`.
