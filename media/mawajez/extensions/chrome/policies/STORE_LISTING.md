# Store Listing Notes

Name: Mawajez News Hub

Short description: Multilingual global news search, source filters, favorites, side panel, alerts, and session restore from Mawajez public feeds.

Single purpose: Provide quick access to Mawajez public news feeds with local favorites, browser side-panel viewing, optional matching-story notifications, and browser session continuity.

Data handling: Favorites, preferences, notification settings, refresh interval, and cached feeds remain in browser storage. The extension fetches public JSON from `https://mawajez.com/api/`.

Owner: RealLexi LLC, https://reallexi.io
