## Public Media

Run `npm run package` to regenerate the extension packages, policies, public files, screenshots, icons, logos, and store media. The package command syncs the generated bundle into the local `ddkits/Reallexi-Public-media-and-files` clone at `media/mawajez`.

Use `npm run publish:public-media` to sync, commit, rebase, and push the public media repo after a fresh package run.
