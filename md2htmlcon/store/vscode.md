# MD2HTML Converter

Short description: Convert Markdown to HTML and HTML to Markdown with editing, diff, preview, and export tools.

Long description: MD2HTML Converter gives VS Code users a polished local converter for Markdown and HTML. Convert the current file or selection, preview sanitized HTML, compare source and output edits, copy results, and export without leaving the editor.

Features:

- Markdown to HTML
- HTML to Markdown
- Auto-detection and manual direction override
- Sanitized preview
- Inline and side-by-side diff
- Export to workspace, file picker, or untitled editor
- Replace current file only after confirmation
- Status bar and explorer/context menu access for Markdown and HTML files
- Markdown and HTML snippets
- Advanced Markdown and HTML editor tools

Privacy statement: MD2HTML Converter works locally in your editor or browser. Your content is not sent to Reallexi servers for conversion.
