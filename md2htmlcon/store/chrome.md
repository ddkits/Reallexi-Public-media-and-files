# MD2HTML Converter

Short description: Convert Markdown to HTML and HTML to Markdown from pages, selections, and pasted text.

Long description: MD2HTML Converter for Chrome lets users convert selected page content, full webpages, raw Markdown pages, or pasted snippets. It includes a polished popup and side panel with editing, preview, diff, copy, and download tools.

Features:

- Convert selected text with a context menu
- Convert the current page
- Pull the current page with exact-origin permission prompts
- Mouse Reader for click-to-capture page elements
- Detect raw Markdown pages
- Paste Markdown or HTML
- Advanced Markdown and HTML editors
- Download `.html`, `.md`, `.diff`, and `.patch`
- Local-first conversion

Privacy statement: MD2HTML Converter works locally in your editor or browser. Your content is not sent to Reallexi servers for conversion.
