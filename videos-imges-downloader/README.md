# Videos Imges Downloader Screen Recorder Media

This folder contains public-facing media and hosted-policy source files for Videos Imges Downloader & Screen Recorder.

- `icon.png` - README/store icon copied from the shared media icon.
- `icons/` - canonical MV3 extension icons copied into Chrome and Edge builds.
- `privacy-policy.md` - source copy to publish on the publisher website for Chrome Web Store and Microsoft Edge Add-ons submissions.
- `store/` - auto-generated screenshots and promo tiles for extension store submissions.

Place any extension screenshots, screen-recording demo GIFs, and README images for this project in this folder.

Run `npm run icons` to refresh the canonical icon set. Run `npm run build` or `npm run store:assets` to refresh the generated store images.
